# System Requirements

## Functional Requirements:
1. **Sistem dapat menyimpan data nama dan nomor WhatsApp**:
   - Pengguna dapat menyimpan dan mengelola kontak darurat yang mencakup nama dan nomor WhatsApp.

2. **Sistem dapat mendeteksi lokasi pengguna secara otomatis**:
   - Aplikasi menggunakan Geolocation API untuk mendeteksi lokasi pengguna saat tombol darurat ditekan.

3. **Sistem dapat mengirimkan notifikasi darurat ke WhatsApp**:
   - Pengguna dapat mengirimkan pesan darurat beserta lokasi mereka melalui WhatsApp API.

4. **Sistem memiliki antarmuka pengguna yang sederhana namun informatif**:
   - Antarmuka aplikasi dirancang dengan Bootstrap untuk kemudahan penggunaan pada berbagai perangkat.

5. **Riwayat notifikasi darurat dapat dilihat oleh pengguna**:
   - Pengguna dapat melihat riwayat notifikasi darurat yang telah dikirim.

## Non-Functional Requirements:
1. **Keamanan**:
   - Data kontak dan notifikasi harus terlindungi dengan enkripsi dan tidak dapat diakses oleh pihak ketiga.

2. **Responsivitas**:
   - Aplikasi harus berfungsi dengan baik di perangkat desktop dan mobile, memastikan pengalaman pengguna yang konsisten.

3. **Kecepatan**:
   - Pengiriman notifikasi harus dilakukan dalam waktu kurang dari 5 detik untuk memastikan respon cepat dalam situasi darurat.

4. **Availability**:
   - Aplikasi harus tersedia 24/7 tanpa downtime untuk memastikan bahwa pengguna dapat mengaksesnya kapan saja dalam kondisi darurat.

## Minimum System Requirements (Web)

### Operating System:
- **Windows**: Windows 7 atau yang lebih tinggi
- **macOS**: macOS 10.12 atau yang lebih tinggi

### Browser:
- Google Chrome (versi terbaru)
- Mozilla Firefox (versi terbaru)
- Microsoft Edge (versi terbaru)

### Processor:
- Intel Core i3 (1.8 GHz) atau setara

### Memory:
- 4 GB RAM

### Storage:
- 5 GB ruang kosong (untuk penyimpanan aplikasi dan data)

### Display:
- Resolusi 1280x800 atau lebih tinggi

### Internet:
- Koneksi internet broadband untuk akses aplikasi dan pembaruan
