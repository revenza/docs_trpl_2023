Aplikasi ini bernama **Emergency Alert**, dirancang untuk membantu pengguna dalam situasi darurat. Dengan fitur utama seperti pelacakan lokasi real-time dan pengiriman notifikasi darurat ke WhatsApp, aplikasi ini bertujuan menjadi solusi cepat dan mudah digunakan. 

### Pendahuluan

Dalam kondisi darurat, banyak orang mengalami kesulitan untuk mengabari teman atau keluarga dengan cepat. Aplikasi ini hadir untuk menyelesaikan masalah tersebut, dengan kemampuan mengirimkan informasi lokasi langsung ke penerima yang telah terdaftar. Selain itu, fitur pelacakan memungkinkan pengguna mengetahui posisi orang-orang terdekat secara real-time.

Aplikasi ini akan tersedia untuk pengguna Android (versi 7 ke atas) dan iOS (versi 12 ke atas). Semua data pengguna, seperti lokasi dan kontak, akan dienkripsi untuk memastikan keamanan dan privasi.

### Fungsi Utama Aplikasi

Aplikasi ini memiliki tiga fungsi inti yang saling terhubung:
1. Tombol darurat yang bisa mengirimkan pesan dan lokasi langsung ke WhatsApp penerima.
2. Pelacakan lokasi yang memungkinkan pengguna berbagi dan melihat posisi secara real-time.
3. Manajemen kontak darurat untuk menyimpan, mengubah, atau menghapus nomor WhatsApp penerima.

### Cara Kerja

Ketika pengguna menekan tombol darurat, aplikasi akan langsung mengambil lokasi terkini pengguna, kemudian mengirimkannya bersama pesan darurat ke WhatsApp. Lokasi ini juga bisa dilihat di aplikasi jika penerima memberi izin pelacakan. Pengguna dapat mengatur daftar kontak darurat mereka kapan saja melalui fitur CRUD yang tersedia.

### Teknologi yang Digunakan

Aplikasi ini menggunakan **React Native** sebagai kerangka kerja utama untuk mendukung Android dan iOS. Backend aplikasi dikelola dengan PHP, sedangkan data disimpan di MySQL. Untuk mengintegrasikan pengiriman pesan, WhatsApp API digunakan.

### Keandalan dan Kinerja

Aplikasi dirancang untuk bekerja dengan cepat. Notifikasi darurat harus terkirim dalam waktu kurang dari tiga detik. Dalam kondisi normal, aplikasi mampu memproses hingga 1000 notifikasi per detik. Selain itu, backup otomatis dilakukan setiap hari untuk memastikan keamanan data pengguna.

### Kesimpulan

Proyek ini dirancang dengan fokus pada kecepatan, kemudahan penggunaan, dan keamanan. Dengan integrasi teknologi modern, aplikasi Emergency Alert diharapkan bisa menjadi alat yang andal dalam membantu pengguna menghadapi situasi darurat.
